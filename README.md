Java-Calculator-System
Project Description
This is a functional command-line calculator built with Java. It performs basic arithmetic operations based on user-selected operators. I created this project to strengthen my understanding of input parsing and multi-condition logical structures within the Bachelor of Science in Computer Science program.

Core Features
Four-Function Arithmetic: Supports addition, subtraction, multiplication, and division.
Error Handling: Includes logic to prevent division by zero, which is a critical edge case in software development.
Dynamic Input: Uses the char data type to interpret mathematical symbols as commands.

Technical Skills Used
Java Data Types: Managing double for precision and char for operator selection.
Input Management: Implementing the Scanner class to capture numerical and character data.
Conditional Structures: Using nested if-else blocks to route program flow based on user choice.
Logic Safety: Applying validation checks to ensure mathematical correctness.
